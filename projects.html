Evil Hangman

Classic Hangman with a twist! Played through the system's input/output stream, with every guess, the computer changes the word in order to decrease the chances of the user guessing the correctly. Java.

Bignum

A program that represents natural numbers as a reversed list of integers and has procedures to add and multiply. Programmed to be used on large numbers, runs O(n) for addition and o(n^2) for multiplication. Racket.

Eliza

A simple chat bot used through the command line-- Eliza follows a set of rules in order to respond to whatever a user inputs. OCaml.

Rackette

A Racket interpreter that can evaluate simple procedures and lambda expressions. A second version of this project can fully evaluate recursive procedures and lists. OCaml.

Game

A Connect4 game that can be played through the command line with another player or against the coded AI. The user can see the board on the terminal and then chooses which column to place their token in, in order to connect four of their tokens and prevent their opponent from doing the same. OCaml.

Grail Games

A two-player game that makes use of a virtual deck of cards. Similar to the Pokemon card game, duelers are (cards) are either on the bench or in the arena and they fight against opponent duelers using their special fight methods that cause some amount of damage to their opponent. Java.

Contacts

Using a socket-client connection, users may add or remove contacts which are stored in a temporary database of sorts. Users may extract their contacts in XML form as well as start with an XML address book and simply add/remove contacts from there. Java.

Knight Dating

Project that makes use of Binary Search Trees and nearest neighbor search. Nodes store numerical data and a user can interact with a GUI (created using JavaFX) and enter other numerical data and will receive the name of the 'person' whose numerical info most closely matches theirs. Scala.
